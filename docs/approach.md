# Data approach

## DNS

- `dig namigo.com`
- `whois namigo.com`
- Repeat for popular TLDs like `io` and `ai`

## Email

- Lookup: `namigo@gmail.com`
- Repeat for popular providers like `@yahoo.com` and `@outlook.com`
